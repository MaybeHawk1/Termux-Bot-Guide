How To Run A Discord Bot From Termux

(Specifically for python bots)

Step 1. Installing Termux

To install termux you need use one of the following links

  • Termux Github
  • FDroid

DO NOT INSTALL TERMUX FROM THE PLAY STORE, The version used there is outdated

Before doing that you will need to setup the storage access

> termux-setup-storage

Run this a couple times untill you get this prompt

> It appears that directory '~/storage' already exists.
> This script is going to rebuild its structure from scratch, wiping all dangling files.
T > The actual storage content IS NOT Going to be deleted


> Do you want to continue? (Y/N)

Step 2. Installing Python

Once you have installed termux you need to make sure that python is installed

You can do that by using

> pkg install python

Step 3. Transferring The Files

If you're using a file that's on github you can use CURL

Example:

> curl [parameters] [url]

If you're not using github you can download the python file then move it to your downloads folder and then access that folder from termux.

You can move the file to the Home Directory (~) using the mv command

Example: (I'm using main.py as an example here

> mv main.py ~

Step 4. Running The Bot

To run your python files you need to run

> python main.py &

The & sign hides python so you can still use your terminal when running the script

(Do not do this on the first run to make sure there is no configuration needed)

To End The Proccess you need to use the pid your terminal gives you when you run the script with &

Example:

> pkill [PID]

(Without using the square bracket's)

If you ran the script without & you can just use ^C (Control-C)
